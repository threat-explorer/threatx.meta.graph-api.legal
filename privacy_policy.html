Política de Privacidade

Esta Política de Privacidade descreve como coletamos, usamos e compartilhamos suas informações ao usar o aplicativo threatx.search.api.

Coleta de Dados

Coletamos dados pessoais dos usuários, incluindo nome, email, e informações de navegação. Esses dados são coletados para melhorar a experiência do usuário e fornecer serviços personalizados.

Uso dos Dados

Usamos os dados coletados para:

  • Fornecer e melhorar nossos serviços
  • Personalizar a experiência do usuário
  • Comunicar-se com os usuários
  • Realizar análises e pesquisas

Compartilhamento de Dados

Compartilhamos dados com terceiros apenas quando necessário para fornecer nossos serviços ou conforme exigido por lei. Os terceiros incluem:

  • Provedores de serviços
  • Parceiros de negócios
  • Autoridades legais

Base Legal para Tratamento de Dados

Processamos dados pessoais com base nas seguintes bases legais:

  • Consentimento do usuário
  • Execução de contrato
  • Interesses legítimos
  • Conformidade legal

Direitos dos Usuários

Os usuários têm os seguintes direitos em relação aos seus dados pessoais:

  • Direito de acesso
  • Direito de retificação
  • Direito de exclusão
  • Direito de restrição de processamento
  • Direito de portabilidade de dados
  • Direito de objeção

Responsabilidade do Controlador

O controlador dos dados é responsável por garantir a conformidade com as leis de proteção de dados e por proteger os dados pessoais dos usuários.

Alterações na Política de Privacidade

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos os usuários sobre quaisquer alterações significativas.
